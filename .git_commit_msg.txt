fix(pnpm): allow build scripts for prisma, bcrypt, esbuild in Docker

pnpm 11 blocks build scripts by default for security.
Add pnpm.onlyBuiltDependencies to pre-authorize the packages
that need native compilation, avoiding ERR_PNPM_IGNORED_BUILDS
in non-interactive Docker build environments.

- backend/package.json: allow @prisma/engines, bcrypt, esbuild, prisma
- frontend/package.json: allow esbuild
- backend/pnpm-lock.yaml: regenerated with new config
- frontend/pnpm-lock.yaml: regenerated with new config
