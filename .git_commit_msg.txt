fix(docker): upgrade to node:22-alpine for pnpm 11.x compatibility

pnpm 11.x requires Node.js >= 22. The Dockerfiles used node:20-alpine
which caused ERR_UNKNOWN_BUILTIN_MODULE: node:sqlite at install time.

- backend/Dockerfile: node:20-alpine -> node:22-alpine
- frontend/Dockerfile: node:20-alpine -> node:22-alpine
